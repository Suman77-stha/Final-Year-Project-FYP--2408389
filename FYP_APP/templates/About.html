About Trade Vision AI

A Final Year Project developed by Suman Shrestha, focused on building an AI-powered stock market prediction and automated trading system.

About Me

My name is Suman Shrestha, and I developed Trade Vision AI as my Final Year Project. I am passionate about Artificial Intelligence, Machine Learning, and Financial Technology. This project represents my research and implementation of deep learning models applied to real-world stock market data.

What Is This Project About?

Trade Vision AI is a web-based intelligent trading assistant that uses LSTM deep learning models to analyze historical stock market data and predict future price movements. It integrates real-time financial data, AI-based decision-making, and automated trading capabilities.

The system provides Buy, Sell, and Hold recommendations based on predictive modeling and technical analysis indicators.

Technologies Used

Python
Django Framework
LSTM Neural Network
TensorFlow / Keras
REST API Integration
HTML, CSS, JavaScript

Mission & Vision

The mission of this project is to make intelligent stock trading accessible through AI-driven insights and automation. The vision is to create a scalable fintech platform that enhances decision-making using advanced predictive analytics.
